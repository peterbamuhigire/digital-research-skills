# example-market-landscape

Research project workspace.

| Field | Value |
|---|---|
| Project id | `example-market-landscape` |
| Research type | market landscape |
| Audience | internal strategy |
| Variant | executive-report |

## Golden Path

1. Complete `_context/brief.md` and `_context/project-profile.md`.
2. Run research meta-initialization.
3. Execute research waves.
4. Run `python -m engine sync example-market-landscape`.
5. Run `python -m engine validate example-market-landscape`.
6. Build outputs in `05-output/`.
7. Run `python -m engine pack example-market-landscape --out export/example-market-landscape.zip`.
