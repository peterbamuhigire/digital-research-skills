## Notes for QA

1. **LOINC code gaps (95):** Wave 1 agent noted that LOINC's interactive search requires registration; Wave 2 focused on publicly listed codes from LOINC official pages. For exhaustive population, recommend: (a) direct institutional LOINC API access if available, or (b) manual search of `loinc.org/search?q=class:rad` (requires login). BibTeX keys for 9 specific LOINC codes appended; others remain [GAP — no public listing].

2. **RadLex Playbook gaps (130 for RPID + anatomy/finding IDs):** The RadLex Playbook is embedded in a PDF (playbook.radlex.org/playbook-user-guide-2_5.pdf) that is binary and not parseable via WebFetch. RSNA's playbook.radlex.org website does not provide downloadable CSV or XLS export for non-institutional users. Recommendation: contact RSNA directly (radlex-feedback@lists.rsna.org) or seek institutional access. GitHub mirrors do not contain the full RPID inventory.

3. **DICOM SR template gaps (128):** DICOM Part 16 Chapter A exceeds maxContentLength. TID 1500 (Measurement Report) and TID 5200 (Cardiac echo) are documented in literature; others require full Part 16 access. Recommendation: download DICOM Standard PS3.16 PDF directly from NEMA and parse programmatically.

4. **Cobalt-60 / IMRT radiation therapy imaging excluded:** Not clinical diagnostic imaging; out of scope per project instructions.

5. **Interventional radiology procedures partially included:** Arthrography and myelography (with fluoroscopy guidance) included under Fluoro modality at T2 level; vascular interventions (e.g., angiography, ERCP, stent placement) are procedure-grade and may be better handled in the Procedures cohort. For now, marked as T2 (specialist-required, RRH minimum).

6. **Paediatric-specific rows:** All paediatric variants (e.g., CT head dose-reduction, XR chest Image Gently protocol, neonatal hip dysplasia screening) are separate rows per book-derived recommendation #6. Level-of-care marked as HC III/HC IV where performable at primary-referral level, GH/RRH where specialist imaging required.

7. **Contrast safety:** All iodinated-contrast rows carry renal-function and allergy-screen in `preparation_required`; gadolinium rows carry ferromagnetic screening and renal function note. Per evidence discipline, dose levels and contrast agents cited from standard references (ACR practice parameters, Image Gently, RSNA RadiologyInfo).

8. **Sources by tier:**
   - **T1:** LOINC 2.78 (9 codes explicit), ACR practice parameters (anatomy, indications), WHO diagnostic imaging manuals (protocols), RCR iRefer (clinical indications)
   - **T2:** RSNA RadLex documentation (framework, not specific RPIDs), peer-reviewed DICOM SR papers, Image Gently guidelines (paeds doses), radiology teaching literature
   - **T3:** None sole-source; all rows carry at least T1/T2 triangulation

9. **Connectivity tolerance:** All rows default to "online or local DICOM" (PACS integration assumed at GH/RRH); paper-form-equivalent field marks "yes" for all (would require manual cross-walk to Uganda HMIS 105/108 forms in Phase 2 QA).

10. **Radiation dose disclaimer:** Adult doses sourced from ACR practice parameters and RSNA RadiologyInfo; paediatric doses explicitly cite Image Gently or peer-reviewed paediatric radiology sources. All doses represent typical effective dose (mSv); actual dose varies by institution, equipment, and technique.
