## Data Model Notes

**Column definitions (in order):**

- `atc_code` — 5-level ATC code (e.g., L01AA01)
- `atc_level1` — L, M, N, P, R, S, or V
- `atc_level2` — 2-digit therapeutic subgroup (e.g., 01, 02)
- `atc_level3` — 1 letter, pharmacological group
- `atc_level4` — 1 letter, chemical subgroup
- `atc_ddd_value` — numeric Defined Daily Dose (per WHO ATC/DDD Index)
- `atc_ddd_unit` — unit of measurement (mg, g, μg, IU, drops, etc.)
- `rxnorm_rxcui` — RxNorm concept unique identifier (bridge to FHIR Medication.code); [GAP] if not found
- `inn` — International Nonproprietary Name (WHO chemical substance name)
- `who_eml_inclusion` — TRUE (on 23rd EML), FALSE (not on EML), or [GAP — no source found]
- `who_eml_section` — WHO EML section name/number (e.g., "6.2.1 Beta lactam medicines"); [GAP] if not confirmed
- `emhslu_inclusion` — TRUE (on EMHSLU 2023), FALSE (not on EMHSLU), or [GAP]
- `emhslu_vital_essential_necessary` — V, E, N, or [GAP]; per Uganda MoH tier
- `emhslu_level_of_care` — HC II, HC III, HC IV, GH, RRH, NRH, or [GAP]; minimum level of care where drug is listed
- `dosage_forms` — pipe-separated list (e.g., "tablet|capsule|injection|liquid")
- `strengths` — pipe-separated list (e.g., "50mg|100mg|250mg")
- `population` — "adult", "paediatric", "geriatric", "all-ages", or [GAP]
- `paeds_dose_summary` — paediatric dose range (e.g., "0.1–0.2 mg/kg/dose IV") or [GAP]
- `adult_dose_summary` — adult dose range (e.g., "10–20 mg/day oral" or per indication)
- `key_indications` — pipe-separated clinical uses (e.g., "malaria|severe infection|emergency treatment")
- `brand_inn` — INN for branded version (often same as `inn`; included for clarity)
- `brand_1` | `brand_1_holder` — first registered brand name | manufacturer/holder
- `brand_2` | `brand_2_holder` — second brand | holder (if available; else [GAP])
- `brand_3` | `brand_3_holder` — third brand | holder
- `brand_4` | `brand_4_holder` — fourth brand | holder
- `registered_uganda_nda` — TRUE (verified on NDA register), FALSE, [unverified], or [GAP]
- `registered_kenya_ppb` — TRUE, FALSE, [unverified], or [GAP]
- `registered_tanzania_tmda` — TRUE, FALSE, [unverified], or [GAP]
- `controlled_substance_schedule` — Schedule I–V (Uganda Narcotic Drugs and Psychotropic Substances Act) or [GAP]
- `storage` — storage conditions (e.g., "room temperature", "refrigerated 2–8°C", "protect from light")
- `lasa_tallman_form` — LASA (Look-Alike Sound-Alike) tall-man lettering per ISMP (e.g., "DOXOrubicin vs. DAUNOrubicin") [per book-derived clause]; or [GAP]
- `eml_tier` — WHO EML section tier classification for app prioritization (e.g., "core", "complementary", [GAP])
- `level_of_care_min` — HC II, HC III, HC IV, GH, RRH, NRH; minimum cadre/facility needed to dispense/administer safely (per Systems Perspective 2e ch. 11)
- `cadre_min` — CHW (Community Health Worker), Nurse, Midwife, Clinical Officer, Medical Officer, Specialist, Dentist, Pharmacist; minimum cadre per book-derived clause #7
- `code_system_version` — ATC version used (e.g., "ATC/DDD 2024") per Coiera 3e ch. 23
- `code_accessed_date` — date when ATC code was verified (e.g., "2026-05-03")
- `connectivity_tolerance` — "online" (requires connectivity for dosing/interaction checks), "offline" (can run fully offline), or [GAP]
- `paper_form_equivalent` — HMIS form reference (e.g., "HMIS 105 Section 4.2 Drug Therapeutics") or [GAP]
- `coding_rule` — additional coding rules (e.g., "Route of administration may affect ATC code mapping"; per Systems Perspective 2e ch. 12) or [GAP]
- `source_tier` — T1, T2, or T3 (per source-tiers.md)
- `source_citation` — BibTeX key(s) for inline citation (e.g., [who-eml-2023]; [nda-uganda-register-2024])
