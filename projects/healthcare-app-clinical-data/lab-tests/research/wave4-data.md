# Wave 4 Laboratory Tests Data Model

**Date:** 2026-05-03

**Scope:** Laboratory tests — Wave 4 gap-fill for **microbiology, coagulation, endocrinology, molecular diagnostics, cardiac markers, inflammatory markers, toxicology / TDM**. Multi-row-per-test convention employed; each LOINC test may have multiple rows, one per population variant.

**Pass 4 — STG-grounded gap-fill.** Sourced from Uganda IDSR (notifiable disease confirmatory tests), Uganda HIV 2016 guideline (CD4/viral load/cryptococcal antigen), Kenya paediatric reference (age-stratified ranges), and Western fallbacks where local data unavailable.
