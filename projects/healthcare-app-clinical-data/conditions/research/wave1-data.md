## Data Model Notes

**Column definitions (ordered):**
- `icd10_code`: Three-character ICD-10 code (or four/five if subcategory) per WHO 2019 tabular list.
- `icd10_title`: WHO official title from ICD-10 tabular.
- `icd10_chapter`: Roman numeral (I–XXII) per ICD-10 structure.
- `icd10_block`: Block descriptor (e.g., "B50–B64 Protozoal diseases").
- `snomed_ct_concept_id`: SNOMED CT concept ID if mapping exists [GAP if none found].
- `snomed_ct_description_id`: SNOMED CT description ID [GAP — rarely populated].
- `icd11_candidate_code`: ICD-11 code if stable mapping exists [GAP — incomplete at WHO].
- `granularity_caveat`: TRUE if ICD-10 collapses two clinically distinct entities (cite Coiera 3e ch. 23); FALSE otherwise.
- `coding_rule`: Clinical decision rule or tabular exclusion/inclusion note.
- `common_name_en`: Lay/common English name.
- `common_name_local`: Ugandan local language (Luganda / Swahili approximation; for app UI).
- `category_burden`: One of {Communicable, NCD, Maternal-Neonatal, Injury, Mental-Behavioural}.
- `population`: One of {adult, paeds, neonatal, obstetric, geriatric, all}.
- `daly_rank_uganda`: DALY ranking in Uganda (1=highest burden) [GAP for many].
- `daly_rank_kenya`: DALY ranking in Kenya (for triangulation) [GAP for many].
- `daly_rank_tanzania`: DALY ranking in Tanzania (for triangulation) [GAP for many].
- `notifiable_uganda`: TRUE if on Uganda IDSR priority list; FALSE otherwise.
- `clinical_notes`: Brief management note or red flag (max 150 char).
- `ucg_reference`: Uganda Clinical Guidelines 2023 section number [GAP for many].
- `level_of_care_min`: Minimum facility level (HC II / HC III / HC IV / GH / RRH / NRH).
- `cadre_min`: Minimum health worker cadre (CHW / nurse / midwife / clinical officer / medical officer / specialist).
- `code_system_version`: "ICD-10 2019" (per project standard).
- `code_accessed_date`: "2026-05-03" (access date for all codes in this wave).
- `connectivity_tolerance`: "offline-OK" (can diagnose/manage without online systems) or "online-required" (lab/imaging/specialist consultation needed).
- `paper_form_equivalent`: Short description of paper record (RDT card, treatment chart, lab form, etc.).
- `source_tier`: T1 / T2 / T3 (per source-tiers.md).
- `source_citation`: BibTeX key(s) cited for the row (e.g., [who-icd10-2019]; [disease-diagnosis-uganda-2010]).
