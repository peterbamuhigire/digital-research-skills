# example-academic-paper

Research project workspace.

| Field | Value |
|---|---|
| Project id | `example-academic-paper` |
| Research type | paper |
| Audience | academic reviewers |
| Variant | academic-paper |

## Golden Path

1. Complete `_context/brief.md` and `_context/project-profile.md`.
2. Run research meta-initialization.
3. Execute research waves.
4. Run `python -m engine sync example-academic-paper`.
5. Run `python -m engine validate example-academic-paper`.
6. Build outputs in `05-output/`.
7. Run `python -m engine pack example-academic-paper --out export/example-academic-paper.zip`.
