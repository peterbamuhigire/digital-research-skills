## Kenya — Kenyatta University

## Verification protocol

Before using any of these reputations:
1. Check Google Maps reviews (not yet aggregated)
2. Check current Facebook university housing groups for fresh complaints
3. Search "[hostel name] complaint" on local news sites
4. Confirm management hasn't changed since the cited incident (Olympia 2019, Cotilde 2021, Pearl View 2014 are all multi-year-old)
