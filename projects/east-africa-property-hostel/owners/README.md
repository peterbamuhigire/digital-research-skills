## Top findings

- **30+ MUBS students evicted by hostel owners** because the government hadn't paid allowances for 3 months
- Kenyan developers face **18–22% interest rates** vs **4–8% for foreign developers**
- Uganda combined tax stack ≈ **35–40% of revenue** (KCCA Property Rates + Local Hotel Tax + URA Rental Income Tax + VAT + Corporate Income Tax + staff NSSF/NHIF)
- **Acorn Holdings** is now the institutional benchmark — **20,000 beds, KSh 1.52bn FY'25 profit, B1-rated KSh 4.3bn green bond, 99 university partnerships**
- **Currency risk quantified** — KSh -26.8% vs USD added Sh809bn to Kenya's external debt; construction materials +30–50%
- **Kenya National Building Code 2024** post-Sept 2024 fire — 348 boarding schools ordered day-only; explicit dorm rules now enforced

See `research/pain-points-report.md` for the full corpus.
