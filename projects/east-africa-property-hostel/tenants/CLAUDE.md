# CLAUDE.md — tenants sub-project

Sub-project-specific notes. Inherits all rules from the project `CLAUDE.md` and engine root `CLAUDE.md`.

## Cohort definition

Ordinary working adults, families, young singles, retirees in East Africa renting:

- Single-family rentals (bungalows, standalone houses)
- Apartments / flats / "self-contained" units
- Single rooms / bedsitters / *muzigos*
- Boys' quarters / outbuildings
- Condos / gated-community units

## Out of scope

- Students and hostel residents (lives in sibling `students/`)
- Hotels, short-let / Airbnb
- Commercial leases
- LGBTQ+ topics — engine-wide exclusion
