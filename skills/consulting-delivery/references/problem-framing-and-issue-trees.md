# Problem framing and issue trees

## Minimum framing statement

`Should <owner> do X, Y, or Z by <date> in order to achieve <outcome> under <constraints>?`

## Issue-tree rules

- Top level must be mutually exclusive.
- Branches must collectively answer the parent question.
- Use the tree to expose what must be analyzed, not to decorate slides.

## Hypothesis rule

Every major branch should carry an initial view of what is likely true and what evidence would disconfirm it.
