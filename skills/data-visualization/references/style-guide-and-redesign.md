# Style guide and redesign

## Style-guide minimum

- Color palette and semantic use of accent color
- Typeface choices
- Number formatting
- Annotation style
- Source-line format
- Gridline and axis rules

## Redesign workflow

1. State the intended message
2. Identify what the current visual obscures
3. Remove non-data ink that does not help
4. Rebuild with the right chart family
5. Add annotation where the reader's eye should land
