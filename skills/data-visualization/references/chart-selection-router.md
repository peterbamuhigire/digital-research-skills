# Chart-selection router

## Choose by analytic task

- **Comparison** — bar or column
- **Time** — line
- **Distribution** — histogram, box, density, sorted dot plot
- **Geospatial** — map only when location matters more than rank
- **Relationship** — scatter or bubble when correlation is the message
- **Part-to-whole** — stacked bars before pie unless there are very few parts
- **Qualitative** — matrix, annotated list, swimlane, process diagram
- **Table** — when exact values matter more than pattern
