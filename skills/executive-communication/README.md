# executive-communication

Output craft for executive-grade artefacts. Encodes the Minto Pyramid Principle, McKinsey ghost-deck, Bain answer-first, and Zelazny chart-selection.

## Files

- `SKILL.md` — the entry skill (load first)
- `CLAUDE.md` — Claude-specific instructions
- `AGENTS.md` — generic-agent instructions
- `references/pyramid-principle.md` — full framework: governing thought + MECE + 3-rule + skim-path
- `references/scqa-opener.md` — Situation / Complication / Question / Answer pattern
- `references/action-titles.md` — how to convert topic labels to takeaway sentences
- `references/ghost-deck-pattern.md` — McKinsey draft-before-design discipline
- `references/zelazny-chart-selection.md` — five chart families and when to use each
- `references/executive-summary-template.md` — ≤ 300-word template with worked example

## When to use

Any artefact with a senior reader: executive summary, one-pager, slide deck, briefing note, cover memo, proposal. Not for raw research, not for academic writing, not for social media.

## Companion skills

`report-and-proposal-craft` (scaffolding), `professional-word-output` and `python-document-generation` (rendering), `academic-writing` + `academic-reporting-standards` (academic outputs), `analytic-tradecraft` (preserves estimative discipline), `source-evaluation` (preserves citation chain).
