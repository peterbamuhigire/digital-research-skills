# Generic-agent — executive-communication

This skill is portable to Codex and other generic agents. Same five rules apply:

1. Answer first (governing thought in first 50 words)
2. SCQA opener (Situation / Complication / Question / Answer)
3. MECE decomposition (3 sub-points; mutually exclusive; collectively exhaustive)
4. Action titles (complete declarative sentences)
5. One message per page; right chart for the message (Zelazny five families)

Read `SKILL.md` for the decision router and ship-gate. Load the matching references.

If the runtime cannot dispatch sub-agents, run the steps sequentially: read upstream research manifest → identify governing thought → draft skim-path of action titles → fill in evidence → final ship-gate.
