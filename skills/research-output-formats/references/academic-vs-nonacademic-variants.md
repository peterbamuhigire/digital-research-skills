# Academic vs non-academic variants

Every format in this skill has at least one academic variant and at least one non-academic variant. The variant axis is **independent of the format axis** and changes the following dimensions:

| Dimension | Academic variant | Non-academic variant |
|---|---|---|
| **Primary reader** | Specialist peers, examiners, scholarly community | Decision-makers, buyers, voters, general public, clients |
| **Truth standard** | Evidence + peer review + replicability | Evidence + verifiability + accountability of named author |
| **Citation density** | High — every claim cited inline | Low to moderate — sources at the end, hyperlinks, "sources" sidebar |
| **Citation style** | Discipline-prescribed (APA, Chicago, OSCOLA, Bluebook, Vancouver) | House style, hyperlinks, or short reference list |
| **Voice** | Third person, hedged, formal | Variable — first / second / third person, plain or marketing register |
| **Hedging** | Mandatory; over-claiming penalised | Strategic; clarity often beats hedging |
| **Originality claim** | Required, especially in dissertations and papers | Not required; informational or persuasive intent dominates |
| **Length norms** | Set by genre and journal | Set by reader attention and channel |
| **Design / typography** | Conservative; structure carries the work | Active; layout, charts, pull-quotes carry argument |
| **Review regime** | Peer review, examiner, supervisor | Editor, legal review, brand / compliance, regulator |
| **Liability surface** | Reputational and academic-integrity | Reputational, contractual, advertising-standards, professional-conduct, regulatory |
| **Failure mode** | Rejection, retraction, examiner failure | Reader bounce, credibility loss, regulatory action, professional sanction |

## Variant-by-format quick map

| Format | Default academic variant | Default non-academic variant | Notes |
|---|---|---|---|
| Essay | Examiner essay (university coursework, gen-ed) | Long-form magazine essay | Same argumentative spine; voice and citation density change. |
| Paper | Peer-reviewed journal article (IMRaD or thematic) | Industry research note / working paper | Working papers often pre-print versions of papers. |
| Thesis | Master's thesis | Professional capstone / chartered-institute project | Non-academic variant rare; usually still institution-validated. |
| Dissertation | Doctoral dissertation | None native | "Dissertation" in non-academic settings usually means a long report; reject the loose usage. |
| Research proposal | Funding / ethics / supervisory proposal | Consulting or grant proposal | Same scaffolding; "deliverables" replaces "contribution to knowledge". |
| Literature review | Narrative / systematic / scoping / meta-analysis | Landscape scan / state-of-the-field memo | PRISMA reserved for the systematic academic variant. |
| Pamphlet | Scholarly pamphlet (long polemical essay as standalone booklet) | Public-information / advocacy / marketing pamphlet | Non-academic is the default. |
| Op-ed / commentary | Scholarly commentary in journals (Nature, BMJ, Foreign Affairs) | Newspaper / magazine / Substack op-ed | Same shape, different citation regime. |
| Policy brief | Evidence brief (academic policy centres) | Ministerial / NGO / think-tank policy brief | Recommendation must be unhedged in both. |
| White paper | Working paper at academic institute | Vendor / association / think-tank white paper | Vendor pitch belongs at the end, never in the body. |
| Market research report | Academic industry-economics paper | Gartner-style industry report | Different beasts; do not import academic hedging into the commercial form. |
| Market analysis | None — applied by definition | Consulting / corp-dev / VC analysis | Closest scholarly form is a case-based industry paper. |
| Product description | None native | Commerce / catalogue / packaging copy | Regulated categories carry mandatory disclaimers. |
| Case study | Yin / Stake research case study | Marketing case study, teaching case, clinical/legal case | Teaching cases withhold the answer; research cases generalise carefully; marketing cases sell. |
| Feasibility study | Engineering / planning thesis chapter | Donor- or sponsor-funded feasibility study | Sensitivity analysis is mandatory in both. |
| Intelligence report | Strategic / area studies paper | IC / corporate-intel / OSINT assessment | ICD 203 / 206 govern the non-academic form. |
| Legal opinion | Doctrinal analysis / case comment | Client memorandum / formal opinion letter | Pinpoint cite discipline applies in both. |

## Decision rules

1. **Identify the named reader.** If you cannot name the reader, you cannot pick a variant.
2. **Match truth standard to reader.** If the reader does not accept "peer-reviewed" as proof, do not use academic citation density. If the reader does not accept named-expert assertion, do not use the non-academic variant.
3. **Lock voice before drafting.** Mid-document register changes are the single most common reason a piece of research lands badly.
4. **Citation regime follows variant, not topic.** A pamphlet about a cited fact still has a "sources" block. An academic essay about a contemporary product still uses the discipline's citation style.
5. **Format and variant must agree on length.** A 3,000-word "policy brief" is a paper. A 700-word "white paper" is a leaflet. If length and format disagree, you have picked the wrong format.

## Hedging calibration

Calibrate words to evidence in **every variant**. Hedging is not optional in the non-academic variant — it is **moved**: from inline qualifiers to a methods sidebar, a confidence statement, or a footnote.

| Evidence strength | Academic phrasing | Non-academic phrasing |
|---|---|---|
| Replicated, large-N, peer-reviewed | "is" / "demonstrates" | "is" |
| Single study, large-N | "indicates" / "suggests" | "shows" + footnote with caveat |
| Multiple smaller studies, mixed | "appears to" / "may" | "evidence suggests" + sidebar |
| Single study, small-N or contested | "tentatively suggests" / "preliminary evidence" | Avoid in non-academic unless flagged as preliminary |
| Inference from multiple sources | "it can be argued that" + "(synthesis)" mark | "On balance" / "Taken together" + footnote |
| Author judgment | "I argue" (where first person allowed) / "this paper argues" | "We assess" / "Our view is" + reasons |

## When the variant is mixed

Some deliverables sit on the boundary:

- **Practitioner-academic journals** (e.g., *Harvard Business Review*, *MIT Sloan Management Review*, *Stanford Social Innovation Review*) — academic-rigour evidence with non-academic voice. Treat as a non-academic variant *with academic-grade evidence*.
- **Government / institutional reports** (World Bank, UNICEF, AU Commission) — non-academic voice, near-academic citation density, peer-grey-literature review.
- **Court-ready expert reports** — non-academic in voice, but evidence and methodology must withstand cross-examination; treat with academic discipline.

For these, write in the non-academic voice but lift citation discipline and methodological transparency from the academic variant.
