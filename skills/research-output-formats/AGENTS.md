# Agents-specific — research-output-formats

For Codex and generic agents.

- Run this skill before drafting any deliverable that is not already pinned to a template or statutory form.
- Use the selection router in `SKILL.md` to identify the format. Do not draft against a vague brief like "report" or "study".
- After format is fixed, load `references/academic-vs-nonacademic-variants.md` and state the variant in the working header.
- Load only the family reference relevant to the deliverable.
- When delegating to another agent, name the format **and** variant verbatim in the brief.
- The `source-evaluation` evidence-discipline clause applies to every variant of every format.

See `SKILL.md`.
