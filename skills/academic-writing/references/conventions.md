# Academic writing conventions

Academic work is structurally constrained in ways popular writing is not. This skill encodes the conventions that distinguish a publishable thesis / paper / dissertation from a well-researched but unstructured manuscript.

## The four-question filter

Before writing, answer:

1. **Type** — thesis / dissertation / paper (each has different scope expectations)
2. **Variant** — academic (peer-reviewed audience, strict conventions) or popular (general audience, conventions relaxed but rigor preserved)
3. **Discipline** — sciences / social sciences / humanities (each has different IMRAD vs IBC vs essay structure expectations)
4. **Citation style** — APA / Chicago / MLA / Harvard / Vancouver (governs in-text + bibliography form; see `academic-citation-styles`)

Skipping any of these produces work that doesn't fit anywhere.

## Type definitions (typical)

| Type | Length | Audience | Scope | Original contribution |
|---|---|---|---|---|
| **Paper** | 4,000–10,000 words | Field specialists | Single argument or finding | Yes — small to medium |
| **Master's thesis** | 15,000–40,000 words | Examiners, advisor, future researchers | Single research question, narrow | Yes — demonstrable |
| **PhD dissertation** | 60,000–100,000 words | Doctoral committee, field | Substantial original contribution | Yes — significant |
| **Honours thesis** | 8,000–20,000 words | Examiners | Single research question | Modest |
| **Literature review (standalone)** | 4,000–12,000 words | Field specialists | Synthesis of existing work | No (synthesis is the contribution) |

Variants:
- **Academic** — written for the field, full conventions, peer-review-ready
- **Popular** — written for general or adjacent-professional audience, relaxed conventions, accessible language, rigour preserved

## Standard structures

### IMRAD (sciences, quantitative social sciences) — paper / thesis

1. Title
2. **Abstract** (150–300 words; structured: background, methods, results, conclusions)
3. **Introduction** (problem, gap, hypothesis / research question, contribution)
4. **Literature Review** (sometimes folded into Introduction)
5. **Methodology** (design, sample, instruments, procedure, analysis plan, ethics)
6. **Results** (findings, no interpretation)
7. **Discussion** (interpretation, comparison, limitations, implications)
8. **Conclusion** (summary + future work)
9. **References / Bibliography**
10. **Appendices** (instruments, data, transcripts)

### Humanities essay structure — paper / thesis

1. Title
2. **Abstract** (sometimes optional)
3. **Introduction** (problem + thesis statement + chapter map)
4. **Literature Review / Historiography**
5. **Body chapters** (numbered, thesis-driven argument)
6. **Conclusion**
7. **Bibliography**
8. **Appendices**

### PhD dissertation (general)

1. Title page
2. Abstract (often 350 words)
3. Acknowledgements
4. Table of contents, list of figures, list of tables, list of abbreviations
5. **Chapter 1 — Introduction** (problem, research questions, contribution, scope, structure)
6. **Chapter 2 — Literature Review**
7. **Chapter 3 — Methodology**
8. **Chapters 4–7 — Findings / Empirical chapters** (often 3–4 chapters of substance)
9. **Chapter 8 — Discussion** (synthesis across findings)
10. **Chapter 9 — Conclusion** (contribution, implications, limitations, future work)
11. References
12. Appendices

## Hypothesis & research question framing

- **Research question** — open, exploratory ("What factors shape X?")
- **Hypothesis** — testable, predictive ("X is positively associated with Y")
- **Null hypothesis** — the no-effect baseline ("X has no association with Y")
- **Alternative hypothesis** — what you'd accept if null is rejected

Decision rule: **only frame hypotheses if you intend to test them statistically.** Otherwise use research questions.

## Methodology section essentials

Every academic methodology section answers:

1. **Design** — qualitative / quantitative / mixed; cross-sectional / longitudinal / case study / ethnography / experiment
2. **Sample / participants** — n, recruitment, eligibility, demographics
3. **Instruments** — surveys, interview guides, observation protocols, with validation history
4. **Procedure** — what happened in what order
5. **Analysis** — statistical tests, coding scheme, software (SPSS, R, NVivo, MAXQDA)
6. **Ethics** — IRB / ethics committee approval reference; informed consent; data protection; vulnerable populations protections
7. **Reflexivity** (qualitative) — researcher position vis-à-vis subject

## Ethics / IRB statements

Required for any research involving human or animal subjects:

- Approval reference (board, project number, date)
- Informed consent procedure
- Confidentiality / anonymity protections
- Data retention and disposal plan
- Conflict-of-interest disclosure

For East African contexts:
- **Uganda:** UNCST (Uganda National Council for Science and Technology) registration + REC approval
- **Kenya:** NACOSTI permit + KEMRI / institutional ERC approval (where relevant)
- **Tanzania:** COSTECH permit + relevant institutional review
- **Rwanda:** RNEC approval

## Limitations section — non-negotiable

Every academic output names its limitations explicitly:
- Sample size / generalisability
- Self-reporting / response bias
- Recall bias (for retrospective work)
- Methodological constraints
- Time / scope constraints
- Researcher positionality (qualitative)

A paper without a limitations section is presumed to be hiding them.

## Academic vs popular variant rules

| Aspect | Academic | Popular |
|---|---|---|
| Tone | Formal, third-person | Engaging, can be first / second-person |
| Jargon | Field-standard, defined | Avoided or explained in plain language |
| Citations | Full style (APA / Chicago / etc.) | Hyperlinks + endnotes; often less formal |
| Hedging | Heavy ("the data suggest", "it may be that") | Lighter; clear claims with caveats |
| Visual elements | Tables + numbered figures with captions | Infographics, callouts, pull-quotes |
| Length | Word-count expectations rigid | Flexible |
| Audience | Field specialists | General educated reader or specific non-academic professional audience |
| Peer review | Required | Editorial review |

**Both variants must preserve the underlying rigour.** Popular ≠ sloppy; popular = accessible.

## Reflexivity statement (for qualitative / interpretive work)

Required by most journals doing qualitative or interpretive research:

```
The researcher is a [identity, positionality]. Fieldwork was conducted in [language] with the assistance of [translator / interpreter, if applicable]. The researcher's position vis-à-vis [community / subject] may have shaped [data collection / analysis / interpretation] in [specific way]. To mitigate this, [strategies used].
```

## Plagiarism prevention

- Quote directly when using >5 consecutive words from a source
- Paraphrase requires citation, not just rephrasing
- "Common knowledge" exception is narrow — when in doubt, cite
- Self-plagiarism is real — citing your own prior work is correct, not optional
- Run final draft through Turnitin / iThenticate before submission

## Peer review readiness checklist

Before submitting:

- [ ] Abstract under journal limit
- [ ] Keywords selected (typically 3–6)
- [ ] All claims cited
- [ ] All cited sources in reference list (no orphans)
- [ ] All reference-list entries cited (no padding)
- [ ] Tables and figures numbered, captioned, referenced in text
- [ ] Methodology answers all 7 essentials above
- [ ] Limitations section included
- [ ] Ethics statement included if applicable
- [ ] Reflexivity statement included if qualitative
- [ ] Citations in target style (APA / Chicago / MLA / etc.)
- [ ] Author affiliations + ORCIDs
- [ ] Acknowledgements + funding statements
- [ ] Conflict-of-interest declaration

## Anti-patterns

- Skipping the lit review because it's tedious
- Calling exploratory questions "hypotheses"
- Hiding limitations in the conclusion
- Citing sources you haven't read
- Treating popular variant as license for sloppiness
- Inflating word count with restated arguments
- Using passive voice as a default ("It was found that...") — modern style allows active voice in most disciplines
- Treating LLMs as co-authors
