# Claude-specific — academic-writing

- Pick output type and variant before drafting; refuse to mix types.
- Run the source-away workflow on every academic artifact: extract → gate → compose → originality check.
- Notes must be fragments, never sentences (Trzeciak).
- For thesis/dissertation, load `references/eco-thesis-craft.md` and apply the four-rule topic gate first.
- Always load `references/citation-styles.md` after the citation style is chosen; never mix styles within one document.
- Run originality + plagiarism checks before delivery; verbatim 7-word runs are blockers.
