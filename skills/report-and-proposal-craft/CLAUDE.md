# Claude-specific — report-and-proposal-craft

- Pick the artifact type before drafting; refuse to mix types.
- Fill the audience grid (decider/influencer/blocker/user/gatekeeper) before the first heading.
- Refuse to draft until X/Y/Z purpose statement is filled.
- Apply the three-way discipline: findings (descriptive), conclusions (interpretive), recommendations (imperative + owned + dated + costed).
- Place the lead on page 1; never bury the recommendation.
- For white papers, gate every section with Graham's mantra; respect flavor-specific vendor-naming rules.
- For business plans, require sensitivity analysis on financials.
- Write executive summary last; verify it stands alone.
