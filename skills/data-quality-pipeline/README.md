# data-quality-pipeline

Single entry skill for tabular data discipline. Pipeline: encoding repair → tidy check → cleaning → outlier panel → merge audit → four-axis quality score → manifest. Six `references/` (Walker quality, Wickham tidy, Chen merge, encoding-and-unicode, anomaly panel, cleaning recipes). Provenance packet (parquet + profile + dq + manifest) as the ship artifact.

For dataset finding, see `dataset-discovery-and-analysis`.

See `SKILL.md`.
