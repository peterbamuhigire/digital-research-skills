# Claude-specific guidance — academic-writing-conventions

- Run the four-question filter before any drafting (type / variant / discipline / citation style).
- Always include a limitations section — a paper without one will be flagged in peer review.
- For East African research, name the relevant ethics body (UNCST, NACOSTI, COSTECH, RNEC).
- Check the peer-review-readiness checklist in `SKILL.md` before final export.
