# Claude-specific — white-paper-craft

- Pick exactly one flavor; refuse to draft without one.
- Apply the mantra to every section as a deletion test.
- For problem/solution: no product/vendor in first 100 words; vendor only in About + CTA.
- Default to one CTA; reject five-CTA close.
- Footnote every load-bearing claim; flag any statistic >3 years old.
- Use only allowed openings; refuse "In today's fast-paced…".
