# Interview protocols

## Design rules

- Start from decisions, not curiosity.
- Build question blocks by theme, from broad to specific.
- Use open prompts first; probes second.
- Write what would count as disconfirming evidence before the interview starts.

## Minimum guide structure

1. Opening and role/context
2. Experience baseline
3. Core topic blocks
4. Contradictions / exceptions
5. Future-state or recommendation prompts
6. Final "what did I miss?" question

## Notes discipline

- Mark direct quotes clearly.
- Mark paraphrases clearly.
- Record role, date, setting, and any access limitation.
