# Qualitative coding and analysis

## Coding sequence

1. First-pass descriptive codes
2. Second-pass pattern codes
3. Contradiction and negative-case pass
4. Theme consolidation
5. Claim drafting with supporting excerpts

## Output pattern

- Theme
- What supports it
- What complicates it
- What it likely means
- What remains uncertain

## Anti-pattern

Jumping straight from notes to "insights" without any visible coding logic.
