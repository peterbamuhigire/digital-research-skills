# Claude-specific — online-legal-research

- Load `references/source-hierarchy-and-authority.md` to fix mandatory vs persuasive weight before searching.
- Load `references/online-research-workflow.md` before issuing search queries; record the query and date.
- Run citator (or manual currency check) on every cited case and every cited statute. No exceptions.
- Pinpoint cite to paragraph (judgments) or section (statutes) — never a generic page reference for an electronic judgment.

- Use `references/legal-analysis-irac.md` to structure every legal-analysis output. Always include counter-analysis.
- Use `references/citation-and-quoting-discipline.md` to govern quotes, paraphrases, and signal use.
- Refuse to invent case names, statute sections, or repository contents. Every authority must be verified in the live repository.
- Embed the evidence-discipline hard-constraint clause **verbatim** in every legal-research sub-agent brief.
