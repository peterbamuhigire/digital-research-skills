# online-legal-research

Method-first skill for online legal research. Jurisdiction-agnostic methodology with an East African overlay (Uganda, Kenya, EAC) for the engine's home jurisdictions.

## Files

- `SKILL.md` — entry, decision flow, reference router
- `CLAUDE.md` — Claude-Code-specific load order
- `AGENTS.md` — Codex / generic-agent equivalent
- `references/source-hierarchy-and-authority.md` — primary vs secondary, mandatory vs persuasive, stare decisis
- `references/online-research-workflow.md` — Boolean / natural-language searches, citator workflow, free vs paid databases
- `references/legal-analysis-irac.md` — IRAC, issue-spotting, counter-analysis, statute interpretation
- `references/citation-and-quoting-discipline.md` — invariant citation rules, signal use, cite-check process
- `references/east-african-overlay.md` — applying the methodology in Uganda, Kenya, EAC, EACJ contexts

## Source books

- Putman & Albright, *Legal Research, Analysis and Writing* (Cengage, 2018)
- Long, *Legal Research Using the Internet*
- *Legal Research* (anonymous compendium)

## Hard rule

No invented case names, statute sections, or pinpoint citations. Every legal authority is verified in the live repository before it ships. The evidence-discipline clause from `source-evaluation` is embedded verbatim in every legal-research sub-agent brief.
