# Claude-specific — reports-proposals-craft

- Refuse to draft until X/Y/Z purpose statement is filled.
- Place the lead on page 1 — never bury the recommendation.
- Reject vague verbs ("consider," "explore") in recommendations; require owner + date + cost.
- Write executive summary last; verify it stands alone.
- Apply SCQA to every persuasive section.
- Run four proofreading passes separately, never combined.
