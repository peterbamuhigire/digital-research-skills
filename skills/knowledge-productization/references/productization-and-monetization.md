# Productization and monetization

## Asset ladder

1. Insight note
2. Internal framework
3. Repeatable report template
4. Workshop / advisory offer
5. Subscription update or dashboard
6. Long-form publication

## Rules

- Package the smallest useful unit first.
- Tie every offer to a real audience pain or decision.
- Keep methodology explicit; trust is part of the product.
- Refresh assets when the evidence base moves.
