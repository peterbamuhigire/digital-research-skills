# systems-thinking-and-mental-models

Router skill with four co-equal toolkits for system, dynamics, reasoning, and judgement work.

## Files

- `SKILL.md` — router; decision tree for which toolkit to load
- `CLAUDE.md` — Claude-Code-specific load order
- `AGENTS.md` — Codex / generic-agent equivalent
- `references/systemigrams.md` — Boardman & Sauser systemigram method
- `references/causal-loops-and-leverage-points.md` — Iceberg, stocks/flows, R/B loops, archetypes, Meadows leverage
- `references/mental-models-catalog.md` — analyst-reasoning checklist of frames
- `references/decision-science-heuristics.md` — intuition, calibration, forecasting routine

## Source books

- Boardman & Sauser, *Systemic Thinking — Building Maps for Worlds of Systems*
- *Thinking in Systems and Mental Models* (anonymous primer; uses Meadows-style framing)
- Brockman (ed.), *Thinking — The New Science of Decision-Making, Problem-Solving, and Prediction*

## Hard rule

Every claim made through these toolkits — every loop, every base rate, every forecast probability — still passes through `source-evaluation`. Method does not exempt method-driven claims from evidence discipline.
