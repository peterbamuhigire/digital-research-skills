# Claude-specific — due-diligence

- Run the CRAWL phases (Collect, Review, Analyse, Weigh, Log) on every engagement.
- Always pair with `source-evaluation`; every finding carries ref + tier + date + confidence + triangulation + CARA-verdict.
- Sanctions/PEP screen requires DOB, nationality, aliases — never name-only.
- Re-run sanctions screens older than 30 days before reporting.
- Pursue UBO to the natural person or flag the opacity barrier explicitly.
- Single-source adverse media is never proven misconduct; report as allegation.
- The engine reports facts; the engine does not give legal advice.
- Use `report-and-proposal-craft` for the report container; apply the three-way discipline.
