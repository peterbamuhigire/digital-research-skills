# Generic-agent — analytic-tradecraft

Portable to Codex and other generic agents. Five rules:

1. ICD 203 nine standards = universal ship-gate
2. Multiple competing hypotheses for any load-bearing judgment (ACH)
3. Kent / ODNI calibrated probability lexicon (verbal term + numeric band)
4. Probability of judgment ≠ confidence in source — state both separately
5. Pre-ship cognitive-bias audit (which biases; what mitigation)

Read `SKILL.md` for the decision router and ship-gate. Load the matching references.

If the runtime cannot run sub-agents, run techniques sequentially: Key Assumptions Check first → ACH on the contested judgment → Devil's Advocacy if consensus is strong → Pre-Mortem before final ship.
