# analytic-tradecraft

Structured analytic discipline for any forward-looking judgment or contested-evidence call.

## Files

- `SKILL.md` — entry skill (load first)
- `CLAUDE.md` — Claude-specific instructions
- `AGENTS.md` — generic-agent instructions
- `references/icd-203-and-tradecraft-standards.md` — ODNI's nine analytic standards as a universal ship-gate
- `references/heuer-pherson-sats.md` — Catalog of structured analytic techniques (ACH, KAC, Devil's Advocacy, Team A/B, Red Cell, What-If, High-Impact/Low-Probability, Pre-Mortem, Indicators-and-Warning, Argument Mapping)
- `references/kent-estimative-probability.md` — Sherman Kent / ODNI calibrated probability lexicon; separation of probability from source-confidence
- `references/cognitive-bias-checklist.md` — 13-bias checklist with one-line mitigations and the SAT to deploy
- `references/sourcing-and-deception.md` — Single-source-risk rule; NATO Admiralty Code; Bruce-Bennett D&D framework; Iraq WMD / Pearl Harbor / Yom Kippur worked cases

## When to use

Any output containing a forward-looking judgment, an "assess / conclude / estimate" statement, a contested-evidence call, or a single-source load-bearing claim.

## Companion skills

`source-evaluation` (runs before), `research-orchestration` (wave plug-in), `executive-communication` (runs after), `academic-reporting-standards` (academic equivalent), `due-diligence` (DD is estimative judgment by another name).
