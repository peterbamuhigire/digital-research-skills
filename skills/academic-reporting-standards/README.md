# academic-reporting-standards

Academic output discipline at Ivy / Oxford / Cambridge / LSE quality. Two layers: Brause practical-craft + EQUATOR formal-reporting.

## Files

- `SKILL.md` — entry skill (load first)
- `CLAUDE.md` — Claude-specific instructions
- `AGENTS.md` — generic-agent instructions
- `references/brause-dissertation-craft.md` — invisible rules, chapter-by-chapter template, examiner expectations, viva preparation
- `references/equator-decision-tree.md` — EQUATOR + PRISMA + CONSORT + STROBE + MOOSE + GRADE + Cochrane + TOP routing
- `references/originality-claim.md` — how to articulate and defend the original-contribution-to-knowledge claim
- `references/methodology-justification-checklist.md` — for every methodological choice, the rationale chain back to theory
- `references/findings-interpretation-criteria.md` — Brause's five tests for findings; data-driven inference rules; raw-data quotation discipline
- `references/viva-defense-preparation.md` — predictable question bank; external-reader reconnaissance; tabbing strategy; biases acknowledgement protocol

## When to use

Doctoral dissertation, master's dissertation, journal article, systematic review, conference paper, research proposal, grant application, book proposal — at Ivy / Oxbridge / LSE-equivalent quality.

## Companion skills

`academic-writing` (citation, plagiarism, originality), `analytic-tradecraft` (estimative discipline inside the discussion), `source-evaluation` (sources tier-rated), `executive-communication` (executive-summary version), `python-document-generation` / `professional-word-output` (rendering).
