# Generic-agent — academic-reporting-standards

Portable to Codex and other generic agents. Five rules:

1. Genre-correct structure (neutral topic headings; not action titles)
2. Original-contribution claim explicit (gap / inconsistency / conflict / theory-test / theory-build)
3. Methodology justified, not described — every choice tied back to theory
4. EQUATOR reporting-standard checklist matched to study type and submitted
5. Examiner-defensible — every paragraph survives viva-style questioning

Read `SKILL.md` for the decision router and ship-gate. Load the matching references.

If the runtime cannot fetch external standards, treat the cached references in this skill as the working baseline; flag any artefact that would need an unfamiliar standard for the user to confirm.
